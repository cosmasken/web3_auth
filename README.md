# `web3_auth`
Web3 Authentication example

## Project Setup
Clone
```bash

git clone https://github.com/cosmasken/web3_auth

```

Deploy to local

```bash

cd web3_auth/

cd web3_auth_frontend/

npm run setup

```

## Running the project locally

Launch Frontend

```bash
# Starts the frontned at localhost:3000/
npm start 

```
